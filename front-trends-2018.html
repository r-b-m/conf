Jeremy Keith


The layers of the web

Sessioon meeldis mulle väga, kuna see oli hästi esitatud ning välja olid toodud webi suuremad põhitõed. Uue tehnoloogia kasutusele võtmist tuleks valida targalt, kuna tulevikus see sama tehnoloogia võidakse tõenäoliselt asendada millegi muu vastu. Samas, kui kaugemale mõelda, siis alumine layer jääb alati, kuna ülemine layer täiustab pidevalt alumist layerit. Tuleks arvestada ning kaaluda pikaajaliste instrumentide kasutamist ehitamisel.

Jeremy Keith


The layers of the web

Layers

Each of the web’s shearing layers can be peeled back to reveal a layer below. Running that process in reverse—applying each layer in turn—is a key principle of resilient web design.
